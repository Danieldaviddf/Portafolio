## 🏗️ Instalación Local
Si quieres explorar el código localmente, sigue estos pasos:

1.  **Clona el repositorio:**
    ```bash
    git clone https://://github.com/Portafolio
    ```
2.  **Instala las dependencias:**
    ```bash
    npm install
    ```
3.  **Ejecuta el proyecto:**
    ```bash
    npm run dev
    ```
